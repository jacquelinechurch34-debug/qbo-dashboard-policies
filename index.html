CFO KPI Dashboard

This is a personal, single-user tool built by Jacqueline Church that connects to her own QuickBooks Online companies via the Intuit Accounting API to generate a private Excel financial dashboard.

It is not a hosted web application and has no public sign-up or login flow. It runs locally as a script on the owner's own computer.

To disconnect this tool from your QuickBooks Online company, go to your QuickBooks Online account, open Settings > Connected Apps, and revoke access from there directly.
