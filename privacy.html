Privacy Policy

Last updated: July 2, 2026

"CFO KPI Dashboard" ("the tool") is a personal, internal-use software tool built by Jacqueline Church to pull financial report data from Jacqueline Church's own QuickBooks Online companies via the Intuit Accounting API, for the sole purpose of generating a private Excel dashboard.

What data is accessed

The tool accesses Profit & Loss, Balance Sheet, Aged Receivables, and Aged Payables report data from QuickBooks Online companies that the account owner has explicitly authorized via Intuit's OAuth 2.0 flow.

How data is used and stored

Data pulled from QuickBooks Online is used only to populate a local Excel workbook for the account owner's own review. OAuth access and refresh tokens are stored locally on the account owner's own computer and are never transmitted to any third party or server other than Intuit's own API endpoints.

No third-party sharing

This tool does not share, sell, or transmit any QuickBooks data to any third party. It is not distributed to, installed by, or used by anyone other than the account owner.

Data retention

Report data is held only in local files on the account owner's computer for as long as the account owner chooses to keep them. OAuth tokens can be revoked at any time from the QuickBooks Online "Connected Apps" settings.

Contact

Questions about this policy can be directed to the account owner, Jacqueline Church.
